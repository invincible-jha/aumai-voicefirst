# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in aumai-voicefirst, please report it
responsibly:

**Email:** security@aumai.dev
**PGP Key:** https://aumai.dev/.well-known/pgp-key.asc

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

**Do NOT open a public GitHub issue for security vulnerabilities.**

## Response Timeline
- **Acknowledgment:** Within 48 hours
- **Initial Assessment:** Within 5 business days
- **Fix Timeline:** Depends on severity (Critical: 72 hours, High: 7 days, Medium: 30 days)

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes |
| < Latest | No |

## Security Practices
- All dependencies scanned via Dependabot
- CodeQL analysis on every PR
- Automated security scanning in CI/CD
- Regular dependency updates
